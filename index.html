Frontend development is the development of graphical user interface of a website, through the use of HTML, CSS and JavaScript, so that user can view and interact with website



  • HTML: The HyperText Markup Language or HTML is the the standard markup language for document designed to be displayed in a web browser.


  • CSS: Cascading Style Sheets or CSS is a a style sheet language use for describing the presentation of a document written in a markup language such as HTML or XML.


  1. Pure CSS

  2. Bootstrap (Framework)

  3. Tailwind (Framework)

  4. Saas (preprocessor)

  • JavaScript: JavaScript often abbreviated JS, is a programming Language that is one of the core technologies of the world wide web, alongside HTML and CSS.It is use to add functionality to website.



  1. Vanila Js

  2. jQuery (Library)

  3. React Js (Framework)

  4. Angular (Framework)

  5. Vue (Framework)

  6. Typescript (Preprocessor)
